AniNumber

AniNumber (Animated Number) is a class that wraps a number and animate it.

Most of the numeric properties of a component is bilza library has AniNumber class. It provides methods like animate, vibrate, jumpBetween etc.

Following methods are available

  1. goto( atSec :number,value :number)
    1. atSec : The time at which to change value.
    2. value : The value (number).
  2. animate( StartSec :number, endSec :number,startValue :number,endValue :number)
    1. StartSec : Start time of animation in seconds.
    2. endSec : End time of animation in seconds.
    3. startValue : Starting value of AniNumber.
    4. endValue : End value of AniNumber.
  3. gotovibrate
    4. Underconstruction
  4. jumpBetween(startSec :number,endSec :number,firstValue :T, secondValue :T,delayInMS :number=1000
    1. StartSec : Start time of animation in seconds.
    2. endSec : End time of animation in seconds.
    3. firstValue : First Value to jump to.
    4. secondValue : Second Value to jump to.
  5. random(StartSec :number,endSec :number,min :number=0, max :number=100, delayInMs :number=10)
    1. StartSec : Start time of animation in seconds.
    2. endSec : End time of animation in seconds.
    3. min : Minimum value of random number.
    4. max : Maximum value of random number.
  6. oscillate(StartSec :number,endSec :number,startValue :number, endValue :number,secPerIter :number= 1,stopAt=endValue
