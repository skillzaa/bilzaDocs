Text Component

Text component as the name suggests is for writing text on the canvas. However this is a very trickey component.

The main reason for the complexity of Text component is that the size of the component is dependent on font size and not directly on the width and height like other components.

Also the width and height of text component do not increase and decrease in proportion to each other, text content can be very wide while the height remains the same.
