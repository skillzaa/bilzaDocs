Core Props

Following is the list of all Core Props that each component inherets from Base Component. Every component has these props but not all implement them and some components implement certain props differently.

border :AniNumber Width of the border
borderContentArea :AniNumber Not implemented yet
color :AniNumber Color of the main content
colorBorder :AniNumber Border Color
colorBackground :AniNumber Color Background
height :AniNumber | AniPercHeight of the comp
opacity :AniNumber Opacity
paddingTop :AniNumber | AniPercPadding Top
paddingLeft :AniNumber | AniPercPadding Left
paddingRight :AniNumber | AniPercPadding Right
paddingBottom :AniNumber | AniPercPadding Bottom
rotation :AniNumber Rotation Angle
showBackground :AniNumber Show/Hide Background
visible :AniNumber Visibility
width :AniNumber | AniPercComp Width
x :AniNumber | AniPercX axis value
y :AniNumber | AniPercY axis value
