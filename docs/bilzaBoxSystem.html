Bilza Box System

Every component (except few like line component) in bilza library is drawn in a box. Following are the components of the box:

  1. The Border

    2. The Border is not included in the size of a component.

  2. The Background

    3. The padding area (the area between the border and content area).

  3. The paddings
    • paddingLeft
    • paddingRight
    • paddingTop
    • paddingBottom
  4. The ContentArea

    5. This is where the actual content are drawn.
